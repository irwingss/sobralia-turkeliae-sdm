# Configuración de datos externos

Este paquete referencia datasets geoespaciales de gran tamaño (decenas de GB)
que **no se incluyen** en el paquete exportado. Descárgalos manualmente desde
las fuentes oficiales y colócalos en las rutas indicadas para que el código
exportado resuelva sin cambios.

## WorldClim 2.1

- Descarga: https://www.worldclim.org/data/worldclim21.html
- Colócalo en: `./data/worldclim`
